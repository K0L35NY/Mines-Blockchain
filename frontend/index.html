Idea

Users answer quiz questions and play a Mines-style game. Correct answers and safe tiles can be rewarded with ERC-20 tokens on a public Ethereum testnet.

Provably fair

The backend generates a secret server seed and sends only its SHA-256 hash to the player. After the game ends, the server reveals the seed so everyone can verify mine positions.

serverSeedHash = SHA256(serverSeed)
mines = f(serverSeed, clientSeed, nonce)
